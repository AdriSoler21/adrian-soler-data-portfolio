# Customer Segmentation & Business Intelligence Platform

## Overview

Proyecto de análisis de clientes desarrollado con Python y Power BI.

El objetivo es analizar comportamiento de compra, rentabilidad, segmentación de clientes y rendimiento de productos para apoyar la toma de decisiones basada en datos.

## Tecnologías

* Python
* Pandas
* Power BI
* Data Visualization
* Customer Analytics
* Business Intelligence

## KPIs Analizados

* Ingresos Netos
* Ingresos Totales
* Clientes Únicos
* Pedidos
* Ticket Medio
* Ratio de Devoluciones

## Funcionalidades

* Segmentación de clientes
* Análisis de clusters
* Rentabilidad por cliente
* Rendimiento de productos
* Evolución temporal
* Dashboard ejecutivo en Power BI
